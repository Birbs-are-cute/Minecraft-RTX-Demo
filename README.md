# Minecraft-RTX-Demo

A clone of Minecraft and Minecraft Ray Tracing at the same time

# Made using:

Unity standard assets,
Blockbench,
CrazyBump,
Unity 2017.4.40f1

# Minimum System Requirments:

Ram 4gb,
Dual Core prossecor,
Intel G41 Express Chipset or better,
Directx 10
